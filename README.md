# my_llm_app

A local LLM playground built with FastAPI and Ollama that lets you chat with multiple open-source models simultaneously. Each prompt is sent to all available models in parallel, the responses are peer-rated by the models themselves in real time, and the best answer is surfaced automatically — with the others available to inspect. Built as a hands-on research sandbox on the path to ML research engineering.

## Architecture

```
Browser (Jinja2 templates)
│
│  POST /api/chat/arena/stream   (SSE)
│  POST /api/chat                (single model, JSON)
│
├── routers/
│   └── chat.py          — FastAPI route definitions
│
├── services/
│   └── llm_service.py   — Ollama calls, arena logic, SSE generator
│
└── Ollama (local)
    ├── llama3.2
    ├── mistral
    └── gemma
```

## Phases

| Phase | Description | Tag |
|-------|-------------|-----|
| 1 | Single-model chat with conversation memory | v0.1 |
| 2 | Multi-model arena: parallel responses + peer voting | v0.2 |
| 3 | Live SSE progress: model status + vote chips update in real time | v0.3 |
| 4 | Streaming token output per model | planned |
| 5 | Experiment tracking + model benchmarking dashboard | planned |

## Quick Start

```bash
# 1. install dependencies
pip install -r requirements.txt

# 2. pull models (requires Ollama — https://ollama.com)
ollama pull llama3.2 && ollama pull mistral && ollama pull gemma

# 3. run
uvicorn main:app --reload
```

Then open [http://localhost:8000](http://localhost:8000).
