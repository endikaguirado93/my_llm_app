# my_llm_app
A local LLM playground built with FastAPI and Ollama that lets you chat with multiple open-source models simultaneously. Each prompt is sent to all available models in parallel, the responses are peer-rated by the models themselves in real time, and the best answer is surfaced automatically — with the others available to inspect. Every session is logged to a local SQLite database, capturing per-model response times and full conversation history as the foundation for future RAG-based model selection. Built as a hands-on research sandbox on the path to ML research engineering.

## Architecture
```
Browser (Jinja2 templates)
│
│  POST /api/chat/arena/stream   (SSE)
│  POST /api/chat                (single model, JSON)
│
├── routers/
│   └── chat.py          — FastAPI route definitions
│
├── services/
│   └── llm_service.py   — Ollama calls, arena logic, SSE generator
│
├── db/
│   ├── database.py      — SQLAlchemy async engine + session factory
│   ├── models.py        — sessions, messages, request_logs tables
│   └── log_service.py   — write helpers (get_or_create_session, log_turn)
│
└── Ollama (local)
    ├── llama3.2
    ├── mistral
    └── gemma
```

## Quick Start
```bash
# 1. install dependencies
pip install -r requirements.txt

# 2. pull models (requires Ollama — https://ollama.com)
ollama pull llama3.2 && ollama pull mistral && ollama pull gemma

# 3. run
uvicorn main:app --reload
```
Then open [http://localhost:8000](http://localhost:8000).

## Logs
Session history is stored in `logs.db` (auto-created on first run). Useful queries:
```bash
# average latency per model
sqlite3 logs.db "SELECT model, ROUND(AVG(latency_ms)/1000, 1) as avg_seconds, COUNT(*) as runs FROM request_logs GROUP BY model;"

# full conversation for a session
sqlite3 logs.db "SELECT role, substr(content, 1, 80) FROM messages WHERE session_id = '<id>' ORDER BY created_at;"

# errored requests
sqlite3 logs.db "SELECT * FROM request_logs WHERE error IS NOT NULL;"
```

Or browse visually with [DB Browser for SQLite](https://sqlitebrowser.org):
```bash
brew install --cask db-browser-for-sqlite
```
